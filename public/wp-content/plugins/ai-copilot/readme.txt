== Installation ==

1. Navigate to the Plugins page in WordPress.
1. Search for "AI Copilot".
1. Click "Install" and experience the AI revolution.

== Usage ==

1. Register an account with OpenAI.
2. Acquire an API key and integrate it into the plugin settings (Dashboard -> AI Copilot -> Settings).
3. Go to AI Copilot admin menu and seve API keys.
4. Always monitor [your OpenAI usage](https://platform.openai.com/account/usage) to keep track.

Upon installation, navigate to Appearance > Menus to incorporate your WordPress menu icons.

== Frequently Asked Questions ==

= What is AI Copilot? =
AI Copilot is a WordPress plugin designed to enhance content creation and management. It integrates AI-driven tools like GPT-4 for automated content generation, SEO improvement, and editor utilities.

= How can I track my API usage? =
The plugin includes a consumption tracker to monitor OpenAI usage, providing insights by user, role, module, and API model. However this is not 100% accutare, is an estimation based on the API calls and the costs detailed in the documentation.

= Is my data secure with AI Copilot? =
Yes, in association with OpenAI, AI Copilot prioritizes data privacy and security. Review OpenAI's Privacy Policy and Terms of Service for details.

= How do I install AI Copilot? =
Install AI Copilot by navigating to the Plugins page in WordPress, searching for "AI Copilot," and clicking "Install."

= How to create OpenAI API =
To create your OpenAI API key you can follow the instructions in our documentation page [here](https://quadlayers.com/documentation/ai-copilot/settings/)

=== Using Third-Party APIs ===

== OpenAI API ==
= Third-Party Service Disclosure =
Our plugin also integrates with the OpenAI API to enhance its functionality. This may include features like content generation, data analysis, or other AI-driven tasks.

= Data Transmission to OpenAI =
When using AI-powered features, certain input data may be sent to OpenAI's servers for processing. We ensure that only necessary data for the feature's functionality is transmitted.

== Pexels API ==
= Third-Party Service Disclosure =
This plugin utilizes the Pexels API for fetching images. When you use features related to image search, your search queries are sent to Pexels to retrieve relevant images.

= Data Transmission to Pexels =
Be aware that search terms used in the image search feature are transmitted to Pexels. No personal data is sent aside from the necessary search queries.

= Legal and Privacy Concerns for Pexels =
For transparency and legal compliance, review Pexels' policies:
